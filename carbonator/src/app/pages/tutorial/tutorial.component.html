Tutorial

  1. Select a pre-defined emission scenario (from the left hand panel). These includes RCP scenarios used by the intergovernmental panel on climate change and a number of other simple test scenarios.
  2. Time series of input variables (forcings) are shown for:
    • CO2 emissions
    • CH4 emissions
    • SO2 emissions (human aerosols)
    • Volcanic aerosol emissions
    • Solar energy at top of the atmosphere
    • Albedo (planet reflectivity)
  3. Individual forcings can be turned off and on (to isolate the influence of particular forcings on the climate system)
  4. Press the RUN SCENARIO button. This will set in motion the model. Many thousands of calculations will be done in the background in the energy balance, carbon cycle and methane models.
  5. Output variables will be displayed.
  6. You can also create your own scenario. Just select the advanced mode.

Advanced Mode

  1. Here you can edit the input timeseries to generate your own scenarios. For most input variable you need to select the number of nodes to edit

  2. You can then drag nodes up and down

  3. For volcanics you have the option to generate random time series of volcanic emissions

  4. In advanced mode you also have the ability to edit model parameters (this should be done with care, altering these parameters can start giving you unrealistic results).
